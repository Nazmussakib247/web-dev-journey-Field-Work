# Array in JavaScript

## Task list:

1. Create an array of 5 fruits and print each fruit using a loop.
2. Use map() to create a new array where each number is doubled [1, 2, 3, 4]
3. Use filter() to get only even numbers from an array.
4. Find the largest number in an array.
5. Create a simple student array object and display each student's name.
Example:
        [ { name: "Rahim", age: 20 },
        { name: "Karim", age: 22 } ]
