# Day 02 — JavaScript Basic Problem Set 

## Variables, Syntax & Types

1. **Create a Variable**  
   Create a variable named `studentName`, store your name, and print it using `console.log()`.

2. **Store Age**  
   Create a variable called `age`, store your age, and print:  
   `I am 22 years old`

3. **Using let**  
   Create a variable `city` using `let`, change its value from `"Dhaka"` to `"Khulna"`, and print both values.

4. **Using const**  
   Create a constant variable `country`, store `"Bangladesh"`, and print the value.

5. **Data Types Check**  
   Create `String`, `Number`, and `Boolean` variables, then print their types using `typeof`.

6. **Simple Addition**  
   Create two variables `a` and `b`, then print:
   - Addition
   - Subtraction
   - Multiplication
   - Division

7. **Operator Practice**  
   Use `+=` operator and print the updated value.

8. **Remainder Operator**  
   Take two numbers and print the remainder using `%`.
