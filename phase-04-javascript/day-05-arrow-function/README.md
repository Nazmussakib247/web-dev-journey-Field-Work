# Arrow Function

## Task list:
1. Convert this function into an arrow function:
function add(a, b) {
 return a + b;
}
2. Create an arrow function that returns the cube of a number.
3. Write an arrow function to check if a number is positive or negative.
4. Create an arrow function that takes an array and returns its total sum.
5.Write an arrow function that converts Celsius to Fahrenheit.
